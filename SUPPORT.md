# Support

Nous Runtime is a community-supported alpha. No professional support or response-time commitment is provided.

For reproducible non-security defects, use the issue tracker for `kicoyini45-blip/nous-runtime` and include the Nous version, operating system, Python version, steps, expected behavior, actual behavior, and redacted logs.

Do not post credentials, private paths, conversation content, runtime state, or security vulnerabilities in public issues. Follow [SECURITY.md](SECURITY.md) for responsible disclosure.

Start with [README.md](README.md), [docs/getting-started/README.md](docs/getting-started/README.md), and [docs/release/KNOWN_LIMITATIONS.md](docs/release/KNOWN_LIMITATIONS.md).
