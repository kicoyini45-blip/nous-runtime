# Nous Runtime Roadmap

Runtime Foundation and Runtime Ecosystem Sprints 13–19 establish Provider onboarding, IDE integration, Python and TypeScript SDKs, Desktop control, Connectors, developer tooling, and public documentation.

After product feature freeze, only blocker fixes, performance, compatibility, installation, security, and documentation corrections are accepted before the v0.1.0 Alpha release candidate. Release validation, packaging, public-tree verification, soak testing, Git history creation, tagging, and publication are separate maintainer-controlled gates.
