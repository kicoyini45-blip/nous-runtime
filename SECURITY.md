# Security Policy

Report suspected vulnerabilities privately to the maintainers through GitHub Security Advisories. Do not open a public issue containing credentials, exploit details, private data, or approval-bypass instructions.

Nous fails closed at governance boundaries. Plugin and Connector actions must declare capabilities and permissions. Provider and Connector credentials must use environment or secret-store references. The Server Runtime remains authoritative; Desktop, IDE, Mobile, and terminal clients are control surfaces.

The Alpha does not claim a complete OS sandbox. See `docs/security/` for trust boundaries and known limitations.
