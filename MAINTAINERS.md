## Responsibilities

The maintainer reviews changes, handles security reports, preserves Runtime ownership and security boundaries, and manages releases.

Use GitHub Issues for reproducible non-security defects. Use the GitHub security advisory workflow for vulnerabilities. See [SECURITY.md](SECURITY.md).

Public release actions require explicit maintainer approval.
