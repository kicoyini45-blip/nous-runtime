# Getting Started

1. Create and activate a Python 3.10+ virtual environment.
2. Install the package with `python -m pip install -e .`.
3. Run `nous doctor` to inspect local prerequisites.
4. Run `nous` to open the persistent terminal interface.
5. Use `/status`, `/help`, and `/dashboard` before configuring model-backed capabilities.

Use `nous --json` for JSON Lines and piped input, or `nous --quiet` for content-only output. Provider configuration is required for model-backed conversation.

Keep `.nous`, credentials, databases, logs, and local configuration outside source control.
