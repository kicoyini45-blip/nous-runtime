# Announcement Draft

Nous Runtime `v0.1.0-alpha` is available as an early open-source release.

Nous is a long-lived intelligent runtime built to serve people, helping them understand, create, and operate complex systems. The Alpha provides a terminal-first conversation and control experience, governed Runtime execution, persistent conversations and event replay, Provider and Connector integration, Workflow and Plugin contracts, Python and TypeScript SDKs, and MVP VS Code and Desktop clients.

This release is intended for local development, integration evaluation and contract review. It is not an operating-system sandbox. Android and distributed-node paths remain experimental, and externally exposed deployments require operator-managed authentication and transport protection.

Start with the installation and quick-start guides, review the security model and known limitations, and use GitHub Issues or Discussions for reproducible feedback. Security reports should follow `SECURITY.md` rather than public issue threads.
