# Website Launch Plan

A website is optional for the Alpha and is not release-gating.

Initial content:

- the bilingual product positioning from README;
- architecture and security boundary summaries;
- installation and quick-start links;
- the terminal demo;
- known Alpha limitations;
- links to documentation, issues, discussions and the security policy.

Launch only after the GitHub repository and release URLs exist. Use static hosting, no analytics by default, no credential collection, and no claims beyond validated Alpha behavior.
